Predictors of frog survival following translocations - create dataset

Roland Knapp

2022-03-04

The code in this notebook creates a dataset for use in analyses to identify predictors of frog survival following translocation (using translocation_survival_analysis.Rmd). The code is written in distinct chunks to allow review of the output from one chunk before running the next. Users without credentials to access the database will not be able to create several of the raw data files, but can use the saved raw data files to run the remainder of the code. Frog translocation and survival data were generated using code in the SNARL1/cmr-analysis GitHub repository.

Combine frog survival files from cmr-analysis repo

The files accessed by this code chunk are generated by code in the xxxxx_analysis_mrmr.Rmd files in the cmr-analysis GitHub repo. To avoid copying those files to this project and duplicating files as a result, this code chunk accesses those files directly. As such, the file path used will likely need to be changed to reflect local directory structure.

files <- fs::dir_ls("../../cmr-analysis/out/tables", glob = "*ind.csv")
frog_survival <- read_csv(files, id = "path") %>% 
  mutate(site_id = str_extract(path, "(\\d)+"), .after = path) %>%  # extracts numeric string from file path
  rename(years_since_intro = years_since_introduction,
         pit_tag_ref = pit_tag_id) %>% 
  filter(years_since_intro == 1) %>% 
  select(site_id, release_date, years_since_intro, pit_tag_ref, median_survival)
rm(files)

Clean data #1

The same duplicate pit_tag_ref was found in frog_translocation, frog_characteristics, and frog survival datasets. That tag was recorded from frogs released at 70641 and 74976, but tag was subsequently only recaptured at 74976: tag associated with 70641 is incorrect. Recode erroneous tag as a pit_tag_ref not present in database to avoid errors during table joins.

frog_translocation <- frog_translocation %>% mutate(
  pit_tag_ref = replace(pit_tag_ref, site_id == "70641" & pit_tag_ref == "900067000117396", "900067000117397"))
frog_characteristics <- frog_characteristics %>% mutate(
  pit_tag_ref = replace(pit_tag_ref, release_siteid1 == 70641 & pit_tag_ref == "900067000117396", "900067000117397"))
frog_survival <-frog_survival %>% mutate(
  pit_tag_ref = replace(pit_tag_ref, site_id == "70641" & pit_tag_ref == "900067000117396", "900067000117397"))

Add frog, translocation, and site characteristics to frog translocation dataset

frog_characteristics <- frog_characteristics %>% 
  mutate(release_siteid1 = as.character(release_siteid1)) %>% 
  select(-relocate_id, -release_siteid1, -release_date, -type)

frog_survival <- frog_survival %>% 
  select(pit_tag_ref, median_survival)
  
frog_translocation <- frog_translocation %>% 
  left_join(frog_characteristics, by = "pit_tag_ref") %>% 
  left_join(frog_survival, by = "pit_tag_ref") %>% 
  left_join(translocation_sites, by = "site_id")

# Add translocation order
frog_translocation <- frog_translocation %>%  
  distinct(site_id, release_date) %>% 
  group_by(site_id) %>% 
  mutate(order = seq_len(n())) %>% 
  ungroup() %>% 
  inner_join(frog_translocation, by = c("site_id", "release_date"))

Clean data #2

Missing frog characteristics associated with pit_tag_ref = 900067000114499. Replace null values (except bd_load) with values from capture at release site early the next summer. Bd_load not replaced because load could have changed during the >9 months between release and recapture.

Missing frog characteristics associated with pit_tag_ref = 900118001027669. Replace null values with values from capture at release site 1-2 weeks later. Bd_load replaced because load likely would not have changed much in the short time period between release and recapture.

Collection (= donor) site_id 72974 recorded as a distinct site despite being closely associated with 70459. Therefore, merged 72974 into the larger site.

Check if sex is correct.

Sex was recorded for frogs immediately prior to their translocation and should match the sex recorded on recapture at the recipient site. Mismatches are often due to frog sex being mis-identified at the donor site due to typically small sizes of frogs at some of the donor sites.
